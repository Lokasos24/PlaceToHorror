# 👁️ PlaceToHorror Engine (working title)

Un motor de juego especializado en **Terror Atmosférico** (Silent Hill), desarrollado desde cero en C++ con SDL2.

> **Estado del Proyecto:** 🛠️ Fase de Cimientos / Experimental.

## 🚀 Características
- **Arquitectura desacoplada:** Sistema de Engine y Window independientes.
- **Renderizado:** Basado en SDL2 (Hardware Accelerated).
- **Filosofía:** Ligero, eficiente y optimizado para hardware antiguo.

## 🛠️ Requisitos de Construcción
Para compilar este motor necesitas:
- **CMake** (3.10+)
- **C++17** o superior.
- **SDL2** (Librerías de desarrollo).
- **MSVC** (Visual Studio 2022 recomendado).
